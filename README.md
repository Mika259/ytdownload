# ytdownload
Simple youtube downloader

# Requirements
- install requirement
```
pip install pytube
```
- python version 3 and upper
```
sudo apt install python3
```
or
```console
pkg install python3
```
# Usage
<b>
cd ytdownload<br>
python3 ytdw.py
</b>
