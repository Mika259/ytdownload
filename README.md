# ytdownload
Simple youtube downloader

# Requirements
- install requirement
```console
pip install pytube
```
- python version 3 and upper
```console
sudo apt install python3
```
or
```console
pkg install python3
```
# Usage
```
cd ytdownload
```
```
python3 ytdw.py
```
